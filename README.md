# 🪙 FLIP — Premium Coin Toss Game

A highly polished browser-based coin flip game featuring cinematic animations, procedural audio, and juicy visual feedback.

Built to push **game feel**, **micro-interactions**, and **mobile-first web polish**.

🔗 **Live Demo:** https://kragenz.github.io/coin-flip-game/
📦 **Repo:** https://github.com/KragenZ/coin-flip-game

---

## ✨ Features

### 🎮 Core Gameplay

* Heads / Tails prediction
* Win / Loss tracking
* Dynamic streak system
* Recent flip history
* Real-time win rate bar

### 🎨 Visual Polish

* Cinematic 3D coin flip
* Nebula animated background
* Particle burst system
* Shockwave impact effects
* Floating score feedback
* Reactive glow states
* Glassmorphism card UI

### 🔊 Procedural Audio (Web Audio API)

* Synthesized flip whoosh
* Metallic landing thunk
* Win fanfare arp
* Lose descending tone
* UI click feedback
* Sound toggle support

### 📱 UX & Quality

* Fully responsive layout
* Mobile-friendly controls
* LocalStorage save system
* Smooth micro-animations
* Installable-ready structure

---

## 🧠 Tech Stack

* **HTML5**
* **Modern CSS (custom properties, conic gradients)**
* **Vanilla JavaScript**
* **Web Audio API**
* **Canvas API (background + particles)**
